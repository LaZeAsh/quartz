---
title: "HTML Tags"
---
**Use all of the american spelling**
```
<p> - Paragraph Tag
<h#> - Heading tag (# can be 1 - 9)
<img> - Image tag
<hr> - Draws a horizontal line (doesn't need closing tag)
<color> - Color tag
<center> - Centers
<iframe> - Inserts a frame of a website / video
<b> - Bold text
<u> - Underline text
<i> - Italicize text 
```
