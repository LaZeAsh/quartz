## What?

A K P Q is the transformations from the original function

## A

A is the amplitude

if a > 1 then vertical stretch
if a < 1 then vertical compression

$\Huge{\frac{max-min}{2}}$ 

## K

if k > 1 then horizontal compression
if 0 < k < 1 then horizontal stretch

$\Huge{\frac{2\pi}{period}}$ 

## P

if p > 0 then translate left
if p < 0 then translate right


## Q

Vertical translation up or down
if q > 0 then translation up
if q < 0 then translation down

$\Huge{\frac{max-amp}{2}}$
